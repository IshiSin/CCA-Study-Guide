The **Claude Certified Architect (CCA)** is Anthropic's foundational certification for practitioners building with Claude. It validates your ability to design, configure, and deploy Claude-based applications across real-world scenarios.

## Domain Weights

The exam is divided into five domains. Questions are distributed according to these weights:

| Domain | Name | Weight | ~Questions |
|--------|------|--------|-----------|
| 1 | Agentic Architecture & Orchestration | 27% | ~16 |
| 2 | Tool Design & MCP Integration | 18% | ~11 |
| 3 | Claude Code Configuration & Workflows | 20% | ~12 |
| 4 | Prompt Engineering & Structured Output | 20% | ~12 |
| 5 | Context Management & Reliability | 15% | ~9 |

<ExamTip>
Domain 1 is the heaviest at 27%. If you're short on time, prioritize Agentic Architecture — understanding multi-agent orchestration patterns, the hub-and-spoke model, and when to use sub-agents versus parallel calls appears frequently throughout the exam.
</ExamTip>

## Scenario Types

The exam includes questions drawn from six real-world scenario categories:

1. **Customer Support Automation** — Multi-turn conversation handling, escalation to human agents, context management across sessions
2. **Code Generation & Review** — Using Claude Code in CI/CD pipelines, generating and validating code, tool integration
3. **Multi-Agent Research** — Orchestrating parallel research sub-agents, aggregating results, handling partial failures
4. **Developer Productivity** — Claude Code configuration, CLAUDE.md hierarchies, custom skills and slash commands
5. **CI/CD Pipeline Integration** — Automated PR reviews, test generation, deployment gating with AI
6. **Structured Data Extraction** — Schema-validated extraction, retry logic, batch processing with the Message Batches API

## Recommended Prerequisites

The exam assumes working familiarity with:

- The Claude API and Anthropic Python/TypeScript SDKs
- Basic agentic patterns (tool use, multi-turn conversations)
- Command-line tools and basic shell scripting
- REST API concepts
