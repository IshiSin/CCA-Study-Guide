---

Summarize what we have discussed so far in this conversation, focusing on what was decided and what still needs to be done. Keep it under 200 words.

Structure the summary as three short sections:

**Goal** — What were we trying to accomplish?

**Done** — What decisions were made, what was built or changed? Include the reasoning for non-obvious choices.

**Remaining** — What is still open, blocked, or not yet started?

Skip filler. If something is obvious from the code context, do not repeat it verbatim.
